# Veloura House

Premium hair salon website built with React and Vite.

## Stack

- React 18
- Vite 5
- React Router
- CSS for layout, premium styling, responsiveness, and motion

## Setup

1. Install Node.js 18 or newer.
2. Run `npm install`.
3. Run `npm run dev`.
4. Run `npm run build` for a production build.

## Pages

- Home
- About
- Pricing
- Gallery
- Contact

## Features

- Premium salon branding with a custom name: Veloura House
- Responsive layout with mobile navigation
- Animated hero and reveal transitions
- Pricing category switcher
- Gallery filtering
- Booking/contact form ready for backend or Google workflow integration

## Note

This environment did not have Node.js or npm installed, so the project structure and source files were created manually and verified with editor diagnostics. Run the install and build commands above on a machine with Node available.
